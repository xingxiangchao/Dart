4. 方法
    1. 方法定义
        `返回类型 方法名(参数1，参数2，...) {
        方法体...
        retuen 返回值
        }`
    2. 方法特性
        1. 方法也是对象，并且有具体类型 Function
        2. 返回值类型，参数类型都可以省略
        3. 箭头语法：=> exper 是 {return exper;} 缩写。只适用于一个表达式
        4. 方法都有返回值，默认 return null 最后一句。
    3. 可选参数
        1. 可选命名参数： {param1, param2, ...}
        2. 可选位置参数： [param1, param2, ...]
    4. 默认参数
        1. 使用 = 在可选参数指定默认值
        2. 默认值只能是编译时常量
    5. 方法对象
        1. 方法可作为对象赋值给其他变量
        2. 方法可作为参数传递给其他方法

    7. 闭包
